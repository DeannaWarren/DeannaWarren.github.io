GIT cheatsheat

  • ref 1
  • ref 2
  • git clone (url) : copy a repository(repo) from online onto your machine
  • git init (name): create a new local repo on your machine
  • git status : shows the new or modified changes ready for staging
  • git add . : stages all changes for commit
  • git add -p (file name) : stages specific changes for commit
  • git commit : creates a save point in the work that can be reverted to
  • git commit -m "(commit message)" : does a commit and allows message to be written inline
  • git (checkout or co) (branch name) : change branches in repo
  • git (checkout or co) -b (branch name): checkout to a new branch
  • git push (location) (branch) : pushes changes to the branch on a different repo, usually to an online version such as GitHub
  • git branch : shows all branches currently open for current repo
  • git branch -d (branch name) : delete a branch that has been merged
  • git pull (location) (branch) : pulls changes from specified branch to current one
